## Calcul pour une action:

### Rentabilité
ri = (Vi - Vi-1)/Vi-1

### Rentabilité moyenne
R = 1/N Σ (ri) 

### Variance
s² = Σ (rt - R)² / n-1

### Risque
s = √s²
