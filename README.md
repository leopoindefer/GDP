## Calcul pour une action:

### Rentabilité à t+1
rt1 = (Vt1 - Vt0) / Vt0

### Variation
rti = (Vti+1 - Vti)/Vti

### Rentabilité moyenne
R = 1/N Σ (rti) 

### Variance
s² = Σ (rti - R)² / n-1

### Risque
s = √s²
