Calcul pour une action: 
Rentabilité à t1 = (Vt1 - Vt0) / Vt0
E(R) = 1/N Σ ((Vn+1 - Vn)/Vn) 
Volatitilé = V((Vn+1 - Vn)/Vn))
Risque = √(V((Vn+1 - Vn)/Vn)))


Calcul pour un portefeuille: 

E(R) = w1R1 + w2R2
